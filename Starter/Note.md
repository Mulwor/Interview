Array.prototype.concat() - используется для объединения двух, или более массивов в один, при этом он не изменяет существующие массивы, а возвращает новый массив.

```
const array1 = ['a', 'b', 'c'];
const array2 = ['d', 'e', 'f'];
console.log(array1.concat(array2));       // ["a", "b", "c", "d", "e", "f"]
```
